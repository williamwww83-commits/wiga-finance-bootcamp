# WIGA Finance - Experiência Digital de Relacionamento Financeiro

Projeto desenvolvido para o Bootcamp da DIO.me, integrando **Inteligência Artificial Generativa**, **Python**, **Análise de Dados** e **UX Design**.

## 🚀 Funcionalidades
1. **Assistente de IA (NLP):** Compreensão de linguagem natural para tirar dúvidas financeiras e sugerir produtos.
2. **Simulador de Investimentos:** Ferramenta interativa de cálculo financeiro com projeções gráficas.
3. **Painel de Dados:** Visualização de transações utilizando Pandas e Plotly.
4. **Persistência de Contexto:** Histórico de conversas mantido durante a sessão do usuário.

## 🛠️ Tecnologias Utilizadas
- Python
- Streamlit
- Pandas & Plotly
- Google Generative AI (Gemini API)
